# Vault Advisor installeren

## Nederlands

Open DIM (https://app.destinyitemmanager.com) en log in. Laat je inventory een keer volledig laden.

Download de zip: https://github.com/Reuzehagel/destiny-vault-advisor -> groene "Code" knop -> "Download ZIP" -> uitpakken.

Chrome / Edge / Brave / Arc:

1. Ga naar `chrome://extensions`
2. Zet "Developer mode" aan (rechtsboven)
3. Klik "Load unpacked" en kies de uitgepakte map (die met `manifest.json` erin)
4. Terug naar DIM, F5, klik een wapen

Firefox:

1. Ga naar `about:debugging#/runtime/this-firefox`
2. Klik "Load Temporary Add-on..."
3. Kies `manifest.json` in de map
4. Terug naar DIM, F5, klik een wapen

Met Firefox moet je dit na elke herstart opnieuw doen, bij Chrome niet.

## English

Open DIM (https://app.destinyitemmanager.com) and sign in. Let your inventory fully load once.

Download the zip: https://github.com/Reuzehagel/destiny-vault-advisor -> green "Code" button -> "Download ZIP" -> unzip.

Chrome / Edge / Brave / Arc:

1. Go to `chrome://extensions`
2. Turn on "Developer mode" (top right)
3. Click "Load unpacked" and pick the unzipped folder (the one with `manifest.json` in it)
4. Back to DIM, F5, click a weapon

Firefox:

1. Go to `about:debugging#/runtime/this-firefox`
2. Click "Load Temporary Add-on..."
3. Pick `manifest.json` in the folder
4. Back to DIM, F5, click a weapon

With Firefox you redo this after every restart, with Chrome you don't.
